Documentation Quick Install
main.rs 
use xeo::Xeo;

fn main() {
    // Initialize
    let mut app = Xeo::new(8080, "0.0.0.0".to_string());

    // Register Routes
    app.route("/".to_string(), "index.html".to_string());
    app.route("/docs".to_string(), "docs.html".to_string());

    // Launch
    app.server();
}

Built for Performance

Every feature designed with zero-cost abstractions and compile-time guarantees.

Blazing Fast

Sub-millisecond latency. Routes are pre-loaded into memory, bypassing disk reads during requests.

Memory Safe

Rust's ownership model eliminates null pointers, data races, and buffer overflows at compile time.

Minimal Dependencies

Built almost entirely on the standard library.

In-Memory Routing

Routes are stored in a HashMap at startup. O(1) lookup speed for every incoming request.

Deterministic I/O

Predictable, synchronous request handling. No massive runtime or hidden background threads.

Raw Control

Direct access to the TCP stream. No middleware bloat hiding the byte stream from you.

Get Started in Seconds

Add Xeo to your Cargo.toml dependencies.

[dependencies] xeo = "0.1.5"

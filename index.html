XEO is a bare-metal Rust framework designed for security engineering. Type-safe, memory-safe, and zero-overhead.

Read Documentation GitHub
main.rs 
use xeo::Server;

fn main() {
    // Initialize a secure server instance
    Server::new("127.0.0.1:8080")
        .secure()
        .get("/", || "Hello from XEO")
        .run();
}

Secure by Default

Automatic protection against common vulnerabilities. Security is baked in, not bolted on.

Bare Metal Speed

Built on top of Hyper and Tokio. Zero-cost abstractions mean you pay only for what you use.

Tiny Footprint

XEO compiles to a single binary with minimal dependencies, perfect for containerized environments.

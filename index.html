Zero Dependencies

No bloat. No hidden costs. Just pure Rust compiled directly to machine code with zero runtime overhead.

Blazing Fast

Sub-microsecond response times. Handle millions of concurrent connections with minimal memory footprint.

Raw TCP Control

Direct socket access. Full control over every byte that enters and leaves your server. No abstractions hiding the metal.

Type Safe

Rust's ownership model guarantees memory safety at compile time. No null pointers. No data races. Ever.

Observable

Built-in metrics and tracing. Know exactly what's happening inside your server at any moment.

Modular

Take what you need. Leave what you don't. Every component is optional and replaceable.

Get Started in Seconds

One command. Zero configuration.

// main.rs
use xeo::prelude::*;

fn main() {
    Xeo::new()
        .route("/", |_req| "Hello, Metal.")
        .bind("0.0.0.0:8080")
        .run();
}

Introduction to XEO

XEO is a high-performance, secure-by-default web framework for Rust. It combines the developer experience of modern frameworks with the raw power of bare metal.

Why XEO?

Building web services in Rust often requires choosing between "easy but slow" or "fast but complex". XEO bridges this gap. It provides:

  • Zero Overhead: Thin wrappers around hyper and tokio.
  • Safety First: Compile-time guarantees against common web vulnerabilities.
  • Productivity: An API that feels familiar to users of Express or Axum.

Quick Start

Add XEO to your Cargo.toml dependencies:

[dependencies]
xeo = "1.0.0"
tokio = { version = "1", features = ["full"] }

Then create your entry point:

use xeo::Server;

#[tokio::main]
async fn main() {
    Server::new("0.0.0.0:3000")
        .get("/", || "Hello World")
        .run()
        .await;
}

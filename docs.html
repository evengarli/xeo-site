Introduction

Xeo is not for everyone. It is a minimal, unopinionated framework designed for developers who want to understand the entire HTTP lifecycle.

Installation

Add the library to your dependencies. No complex CLI tools required.

[dependencies]
xeo = "0.1.3"

Routing

Map raw strings to file paths. The framework loads these into memory immediately.

let mut app = Xeo::new(8080, "0.0.0.0".to_string());

// Register your routes
app.route("/", "index.html");
app.route("/docs", "docs.html");
